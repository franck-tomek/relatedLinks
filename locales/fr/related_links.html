Ce plugin vous permet d'ajouter des liens relatifs à un billet. L'idée est d'inciter le lecteur à poursuivre sa lecture sur le blog avec des billets succeptibles de l'intéresser.

Mode d'emploi

  1. Activer le plugin (onglet Paramètres)
  2. Aller sur la page d'édition d'un billet sous la boîte du contenu.
  3. Cliquer sur "Ajouter un billet à cette liste". Une nouvelle fenêtre s'ouvre (similaire à la fenêtre permettant de faire un lien vers un billet depuis la barre d'outils d'édition). Cliquez sur le lien choisi. Il s'ajoute dans la liste des liens.
  4. Vous pouvez ajouter autant de liens que vous voulez. Vous pouvez les ordonner par glisser/déposer. Il faut ensuite enregistrer.
  5. Vous pouvez supprimer tous les liens liés à un billet.

Template

Ce plugin définit deux templates inc_related_links.html et inc_related_links_with_images.html, que vous pouvez trouver dans le répertoire default-templates du plugin.

Vous pouvez modifier ces templates. Pour cela copiez-les dans le répertoire tpl de votre thème.

Si vous utilisez le plugin themeEditor pour configurer votre thème, vous devriez les voir apparaître dans la liste des templates disponibles et modifiables.

Mise en place

Vous avez plusieurs possibilités pour mettre en place le contenu généré par le plugin.

Automatiquement

Le contenu est mis en place automatiquement en utilisant le comportement publicEntryAfterContent. Vous pouvez modifier le contenu généré automatiquement en modifiant le template inc_related_links.html (ou le template inc_related_links_with_images.html) dans votre thème.

En modifiant le template "post.html" de votre thème

Pour vous aider le plugin définit des nouveaux tags :

  • RelatedLinksIf (bloc) : pour n'afficher le code que si vous avez défini des liens relatifs
  • RelatedLinks (bloc) : boucle sur la liste des liens relatifs.
  • RelatedLinkTitle (valeur) : libéllé du lien
  • RelatedLinkURL (valeur) : URL du lien
  • RelatedLinkImage (valeur) : Image à ajouter au lien

Vous trouverez un exemple de code dans l'onglet "Installation" de la page du plugin.

En incluant le template inc_related_links.html (ou ou le template inc_related_links_with_images.html)

Vous pouvez l'inclure à l'endroit souhaité dans le template "post.html" de votre thème avec le code suivant : 

      {{tpl:include src="inc_related_links.html"}}

Vous pouvez alors soit garder le template inclus tel qu'il est ou alors le modifier dans votre thème.

Avec un widget

Allez sur la page Widgets de présentation et mettez en place le widget "Related Links". Vous pouvez changer le titre du widget.
